1. llama.cpp（llama-server）

  • 配布元: https://github.com/ggml-org/llama.cpp

2. NVIDIA CUDA Runtime（CUDA ビルド利用時のみ）

  • 用途: NVIDIA GPU での推論高速化（CUDA 版 llama.cpp と併せて自動取得）
  • ライセンス: NVIDIA End User License Agreement
  • 配布元: https://developer.nvidia.com/cuda-toolkit

3. Windows App SDK / WinUI 3

  • 用途: アプリケーション UI 基盤（ZIP に同梱）
  • ライセンス: Microsoft Software License Terms
  • 配布元: Windows App SDK

4. .NET ライブラリ

5. SQLite / sqlite-vec（vec0.dll）

  • 用途: RAG のベクトル検索（アプリに同梱）

6. 既定モデル（Google Gemma 4 E2B・初回起動時に自動取得）

  • 用途: 既定のチャットモデル gemma-4-E2B_q4_0-it.gguf（models\ が空で、初回セットアップで自動セットアップを選んだ場合）
  • 初回セットアップでお手持ちの GGUF フォルダを指定した場合、または models\ に既にチャット用 GGUF がある場合は取得しません
  • 利用条件: Gemma Terms of Use に従ってください
  • 取得元: Hugging Face（google/gemma-4-E2B-it-qat-q4_0-gguf）
